# Twitch 賭博計算機 (Gambling Calculator) - 多選項版

這是一個專為 Twitch 實況主觀眾設計的賭博輔助工具。支援目前流行的 **多選項預測** (Channel Points Predictions)，幫助你快速計算期望值 (EV)，並利用凱利公式 (Kelly Criterion) 提供最優下注建議。

## ✨ 主要功能
- **多選項支援**：不限於藍紅兩方，可自行新增選項 (例如 A, B, C, D...)。
- **動態 EV 計算**：輸入各選項的總下注池與勝率，自動找出期望值最高的選項。
- **防呆機制**：
  - 自動檢查總勝率是否為 100%。
  - 最少保留兩個選項。
- **凱利公式資金建議**：提供科學的下注比例，避免「憑感覺」梭哈。
- **Twitch 風格 UI**：深色模式設計，長時間觀看舒適。

## 📖 如何使用
1. **輸入勝率**：手動輸入每個選項預估的勝率 (例如 50%, 25%, 25%)。請確保加總接近 100%。
2. **輸入下注池**：填入每個選項目前的總下注點數 (Total Points)。
3. **新增/刪除選項**：點擊「+ 新增選項」或右側的「🗑️」來調整選項數量。
4. **資金管理**：(選填) 輸入你的總持有點數，計算機將直接告訴你應該下注多少。

## 🚀 如何開啟網頁版
1. 進入此 GitHub 存儲庫的 **Settings (設定)**。
2. 點選左側選單的 **Pages**。
3. 在 **Build and deployment** 下的 **Branch**，選擇 `main` 並點擊 **Save**。
4. 網頁連結將會生成 (例如 `https://NyannNya.github.io/twitchgame/`)。
