# Twitch 賭博計算機 (Gambling Calculator)

這是一個專為 Twitch 實況主觀眾設計的賭博輔助工具。幫助你快速計算期望值 (EV)，並利用凱利公式 (Kelly Criterion) 提供最優下注建議。

## 功能
- **快速計算賠率與勝率**：輸入雙方下注量與勝率，自動計算。
- **EV 期望值分析**：只有當期望值 > 10% 時才建議下注，避免盲賭。
- **凱利公式資金管理**：根據你的持有點數，精確計算應下注金額，最大化長期收益。
- **深色模式 UI**：採用 Twitch 風格設計，長時間觀看也不傷眼。

## 如何使用
1. 輸入 **藍方勝率** (%)。
2. 輸入 **藍方** 與 **紅方** 的總下注點數。
3. (選填) 輸入你目前的 **持有點數**。
4. 查看下注建議與金額。

## 如何開啟網頁版 (GitHub Pages)
1. 進入此 GitHub 存儲庫的 **Settings (設定)**。
2. 點選左側選單的 **Pages**。
3. 在 **Build and deployment** 下的 **Branch**，選擇 `main` 並點擊 **Save**。
4. 等待幾分鐘後，你將獲得一個網址 (例如 `https://NyannNya.github.io/twitchgame/`)，即可隨時隨地使用。
